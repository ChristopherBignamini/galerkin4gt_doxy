Detailed Description

template<typename Geometry, typename... Rest>
struct gdl::aggregator_type_tuple< assembly< Geometry >, Rest... >

struct aggregator_type_tuple

a tuple of assemblers

the purpose of this class is to reduce the boiler plate code on the user side: we define here the arg types we need for computing the quantities used by the assembler, so that the user does not have to deal with them the only thing exposed to the user are the API to retrieve some of the stages (like the the compute_jacobian stage).
